# photon-main

Static Photon web repository generated by `photon_webrepo_bootstrap.py`.

## What this repo contains

- `index.json` — machine-readable repo index
- `packages/` — per-package JSON metadata
- `nuclei/` — published Photon recipe files
- `index.html` — simple browsable landing page

## Notes

- If you publish from a branch, GitHub Pages can use the branch root or a
  `/docs` folder as the publishing source. GitHub also supports publishing
  with a custom GitHub Actions workflow for sites that need a build step.
- This repo is generated so you can use either approach.
