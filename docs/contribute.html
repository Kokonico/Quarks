Contribute Packages

Help expand the Quarks ecosystem by submitting nuclei recipes for packages.

Quick Start: Create a .nuclei file, test it locally with quarks install ./your-recipe.nuclei, then submit via GitHub PR.

Nuclei DSL Reference

Basic Structure

nuclei "package-name", "1.0.0" do description "What the package does" homepage "https://example.com/" license "MIT" category "appcategory" source "https://example.com/package-1.0.0.tar.gz", checksum: "sha256-checksum-here", algorithm: "sha256" build do configure make install "make DESTDIR=${DESTDIR} install" end end

Required Fields

FieldDescriptionExample
namePackage name (lowercase)"gcc"
versionPackage version"13.2.0"
sourceDownload URL + checksumSee examples below
buildBuild commands blockSee examples below

Optional Fields

FieldDescriptionExample
descriptionShort description"GNU Compiler Collection"
homepageProject website"https://gcc.gnu.org/"
licenseLicense (SPDX)"GPL-3.0-or-later"
categoryPackage category"sys-devel"
slotPackage slot"0" or "2"
dependenciesRuntime deps["dev-libs/glibc"]
use_flagsUSE flags["X", "ssl"]
blockersIncompatible packages["sys-devel/gcc:11"]

Build Commands

CommandDescription
configureRun ./configure with common options
makeRun make with parallel jobs
cmakeRun cmake build system
mesonRun meson build system
run "cmd"Run arbitrary shell command
install "cmd"Run install command

Build System

# Specify build system build_system :autotools # ./configure (default) build_system :cmake # cmake .. build_system :meson # meson setup build build_system :make # just make build_system :manual # custom commands only

Dependencies

dependencies [ "dev-libs/glibc", "dev-libs/gmp", "sys-libs/ncurses" ] build_dependencies [ "sys-devel/m4", "sys-devel/binutils" ]

Configure Options

configure \ "--prefix=/usr", "--libdir=/usr/lib64", "--sysconfdir=/etc", "--localstatedir=/var"

Environment Variables

env { "CFLAGS" => "-O2 -pipe", "CXXFLAGS" => "-O2 -pipe", "MAKEOPTS" => "-j$(nproc)" }

Post-Install Scripts

post_install do run "ldconfig" run "gtk-update-icon-cache -q /usr/share/icons/hicolor" run "gdk-pixbuf-query-loaders --update-cache" end

Getting SHA256 Checksum

# Download the source first, then: sha256sum package-1.0.0.tar.gz # Output: abc123... package-1.0.0.tar.gz

Real Examples

Autotools Package (wget)

nuclei "wget", "1.21.4" do description "Network utility to retrieve files from the Web" homepage "https://www.gnu.org/software/wget/" license "GPL-3.0-or-later" category "net-misc" dependencies ["net-libs/gnutls", "dev-libs/libidn2"] source "https://ftp.gnu.org/gnu/wget/wget-1.21.4.tar.gz", checksum: "81542f5cefb8faacc39bbbc6c82ded80e3e4a88505ae72ea51df27525bcde04c", algorithm: "sha256" build do configure make install "make DESTDIR=${DESTDIR} install" end end

CMake Package (cmake)

nuclei "cmake", "3.28.1" do description "Cross-platform make system" homepage "https://cmake.org/" license "BSD-3-Clause" category "dev-util" dependencies [ "dev-libs/libuv", "app-arch/libarchive", "net-misc/curl" ] source "https://github.com/Kitware/CMake/releases/download/v3.28.1/cmake-3.28.1.tar.gz", checksum: "15e94f83e647f7d620a140a7a5da76349fc47a1bfed66d0f5cdee8e7344079ad", algorithm: "sha256" build do cmake make install "make DESTDIR=${DESTDIR} install" end end

Meson Package (ninja)

nuclei "ninja", "1.11.1" do description "Small build system with a focus on speed" homepage "https://ninja-build.org/" license "Apache-2.0" category "dev-util" dependencies ["dev-lang/python3"] source "https://github.com/ninja-build/ninja/archive/refs/tags/v1.11.1.tar.gz", checksum: "31747ae633213f1eda3842686f83c2aa1412e0f5691d1c14dbbcc67fe7400cea", algorithm: "sha256" build_system :cmake build do cmake make install "DESTDIR=${DESTDIR} make install" end end

Package with USE Flags (vim)

nuclei "vim", "9.0.2116" do description "Improved version of the vi editor" homepage "https://www.vim.org/" license "VIM" category "app-editors" slot "0" use_flags ["X", "gui", "perl", "python", "ruby", "tcl"] dependencies [ "lib/ncurses", "dev-lang/python3", "dev-lang/tcl" ] source "https://github.com/vim/vim/archive/refs/tags/v9.0.2116.tar.gz", checksum: "a79783d6e2a46ba446fc695cdb1aa26be511f9db4826aa40b57c89b0392c59ba", algorithm: "sha256" build do run "./configure", "--prefix=/usr", "--enable-gui=gtk3", "--enable-perlinterp", "--enable-python3interp", "--enable-tclinterp", "--with-tlib=ncurses" make install "make DESTDIR=${DESTDIR} install" install "ln -sf vim ${DESTDIR}/usr/bin/vi" end end

Package with SLOT (gcc)

nuclei "gcc", "13.2.0" do description "GNU Compiler Collection" homepage "https://gcc.gnu.org/" license "GPL-3.0-or-later" category "sys-devel" slot "13" dependencies [ "sys-libs/glibc", "dev-libs/gmp", "dev-libs/mpfr", "dev-libs/mpc", "dev-libs/isl" ] build_dependencies [ "sys-devel/m4", "sys-devel/binutils" ] source "https://ftp.gnu.org/gnu/gcc/gcc-13.2.0/gcc-13.2.0.tar.gz", checksum: "...", algorithm: "sha256" build do run "mkdir -p build && cd build" run "../configure", "--prefix=/usr", "--libdir=/usr/lib64", "--enable-languages=c,c++", "--disable-multilib" make install "make DESTDIR=${DESTDIR} install" end post_install do run "ln -sf gcc ${DESTDIR}/usr/bin/cc" end end

Submission Guidelines

  • Test your recipe locally first: quarks install ./your-recipe.nuclei
  • Verify checksums match upstream source exactly
  • Use stable release URLs, not git/nightly unless requested
  • Include all dependencies (both runtime and build)
  • Test build dependencies are available in Quarks
  • Keep descriptions under 100 characters
  • Use correct SPDX license identifiers

How to Submit

Option 1: GitHub Pull Request

  1. Fork the Quarks repository
  2. Add your nuclei recipe to the nuclei/ directory
  3. Include the package in appropriate category subdirectory
  4. Create a PR with package description and build notes

Option 2: GitHub Issue

Open an issue with:

  • Package name, version, and atom (e.g., app-editors/vim)
  • Upstream source URL
  • Your complete nuclei recipe
  • Build/test notes

Recipe Template

nuclei "PACKAGE_NAME", "VERSION" do description "PACKAGE DESCRIPTION" homepage "https://example.com/" license "SPDX-LICENSE" category "CATEGORY" # Runtime dependencies dependencies [ "dev-libs/needed-lib" ] # Build-time dependencies build_dependencies [ "sys-devel/m4" ] source "https://example.com/package-VERSION.tar.gz", checksum: "SHA256_CHECKSUM", algorithm: "sha256" build do configure make install "make DESTDIR=${DESTDIR} install" end end
